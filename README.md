# GX_Project

Prerequisites
1. Python 3.x
2. MSSQL Server

Installation
1. Set up MSSQL "AdventureWorks2012" database locally
2. Create user in MSSQL to connect from Python
3. Install Microsoft ODBC driver for SQL
4. Clone the repository to your local machine
